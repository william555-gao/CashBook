CashBook
一个基于三层架构模式的web记账本项目，正在完善中······


一、项目简介
本项目主要使用了JavaWeb技术，包括Jsp，Servlet，JDBC                                                                                          
项目GitHub地址：https://github.com/zqnh/CashBook                                                                                           
二、项目主要使用的技术                                                                                                                      
 前端
 - 使用了BootStrap框架，展示页面
 - JSP处理前端数据
 - Ajax异步访问（准备实现）                                                                                                                 
 后端
 - Servlet处理业务
 - JDBC连接数据库(Druid)
